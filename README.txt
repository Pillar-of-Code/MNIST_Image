This repository is an image-converted MNIST data set.

A file tree represents type/train/label/file .
A file name is composed of like this example : 540_5.png. 540 represents sequencial number in MNIST. 5 means label.

There are 3 types of dataset: black-background, white-background, Comma-Seperated Values(CSV).

This dataset is just a conversion of the MNIST database by LeCun, Cortes and Burges. http://yann.lecun.com/exdb/mnist/
